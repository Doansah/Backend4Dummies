### Bash is pretty cool 

Bash is the command line interpreter and scripting language used to interact with your Operating System. Bash communicates with your OS kernel to execute commands on your system. 

Here's a few things you can do: 

1) Create Aliases

instead of typing out a long cd command to a deep file path, you can set an alias in your '~/.bashrc' file. 

2) Write Functions! 

I got sick and tired of going through the entire git add, commit, push sequence to make a tiny commit on one file, so I wrote a bash function which simply automates that process for me.
