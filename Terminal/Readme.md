2) Learn some of the syntax 

- Kleene Star * 

The Kleene Star is a placeholder for 'any' 

For instance if you wanted to list all files within a directory called images which end in '.png'. Then you could run the command:

***ls *.png**

-

3) Piping

Piping is the ability to take the output of a the first command and make it the input of the second command. Put differently, its taking the standard out of command 1, and making std input of command 2. 

Lets say you wanted to list all files that have include the text 'dog' in their name. 'ls' is the first command, and 'grep' acts as the second command. 

*** ls | grep "dog"***

### VIM (Goat)
IN PARTICULAR... I really like vim. 

**VIM** is a highly configurble, keyboard-driven text editor. Vim is wonderful because it was designed for you to keep your hands on the keyboard at all times. You can navigate, copy, paste, and search for text all by using key commansd rather than reaching for a mouse. 

If you're a person who's memorized a ton of keyboard shortcuts, and likes typing quickly, VIM is for you!
